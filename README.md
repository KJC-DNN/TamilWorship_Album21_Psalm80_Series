# Album 21 – Psalm 80 Series (6 Songs)

6 powerful Tamil worship songs based on Psalm 80.
Freely given – All glory to Jesus Christ.

Song 1: கர்த்தருக்கு கோபம் மூட்டாதே
Song 2: கறவலாடுகளின் பின்னாகத் திரிந்த
Song 3: உம்மை அறியாத ஜனங்களுக்கு உம்மை அறிவியும் 
Song 4: கேருபீன்கள் மத்தியில் வாசம் செய்யும் கர்த்தரே
Song 5: திராட்சை கொடியை எகிப்திலிருந்து கொண்டுவந்த தேவனே
Song 6: கர்த்தரைக் கெம்பீரமாய் பாடி

ஆமென். அல்லேலூயா! 
Hallelujah forever!
